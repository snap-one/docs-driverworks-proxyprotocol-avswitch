## LOUDNESS\_CHANGED

Loudness state (On/Off) has changed.


### Name

`LOUDNESS_CHANGED ()`


| Parameter | Type | Description                                                         |
| --------- | ---- | ------------------------------------------------------------------- |
| LOUDNESS  | BOOL | True/False                                                          |
| OUTPUT    | INT  | Indicates a change in loudness for the specified output binding ID. |



### Returns

`None`
