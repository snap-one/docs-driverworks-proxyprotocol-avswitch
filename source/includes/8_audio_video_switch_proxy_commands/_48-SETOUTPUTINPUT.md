## SET\_OUTPUT\_INPUT

Specify input and output selection of device.


### Name

`SET_OUTPUT_INPUT ()`


| Parameter | Type | Description                                                |
| --------- | ---- | ---------------------------------------------------------- |
| INPUT     | INT  | Input Binding ID - should be in the BindingID range below. |
| OUTPUT    | INT  | Output Binding ID                                          |



### Returns

`None`
