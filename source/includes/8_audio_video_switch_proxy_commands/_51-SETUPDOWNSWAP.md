## SET\_UP\_DOWN\_SWAP

Swap page UP/DOWN buttons (used by AVGen)


### Name

`SET_UP_DOWN_SWAP ()`


| Parameter | Type | Description                                                                           |
| --------- | ---- | ------------------------------------------------------------------------------------- |
| SWAPFLAG  | BOOL | Swap page UP/DOWN buttons (used by AVGen). `TRUE` to enable swap, `FALSE` to disable. |


### Returns

`None`
